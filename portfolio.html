Portfolio

Check out some of the projects I have worked on. Most of them are somewhere related to the field of Machine Learning, Deep Learning

RU Music

June-2020

Implemented K-NN algorithm to find K similar songs according to the preferences of user. Used subset of data released by Echonest as part of The Million Song Dataset Challenge. It considers different measures of songs like acousticness, liveness, speechiness, loudness etc and finds K similar items from the dataset and recommends those to user. Developed web portal using Spring framework where user can play songs and get recommendations. Worked on REST Web Services, used Mongo DB for database services.

Technologies used:

  • Python/ J2EE - Java Spring.
  • Web Scraping - YouTube
  • git repository

Shortest Path on GoogleMaps using A*

May-2020

Applied A* algorithm in order to find shortest path from given source and destination. Data extracted from OSM, and contains ~27k nodes and ~63k edges. It always tries to find the optimal path. So, in most cases it shows the path which is suggested by Google when you select travelling mode as "Walk".

Technologies and third party APIs used:

  • Python.
  • Google Maps Javascript API.
  • OSM for data generation.
  • git repository

Hand Written Digit Recognition

CS520, Spring-2020

Applied different classification algorithms like Bayesian Network, Perceptron Learning, Neural Netwrok, MIRA on famous MNIST dataset. Developed above mentioned algorithms from scratch. Got accuracy 77.2%, 81.4%, 79.4%, 80.6% respectively. Accuracy is bit low because we were given a small part of original dataset. This project was part of CS520 coursework.

Also developed UI, where user can draw image and model will predict the letter. (Output demo can be found in GIT repo under outcome folder.)

Technologies used:

  • Algorithms and API developed in Python.
  • UI built using HTML, Javascript.
  • Connection between front-end and back-end using AJAX.
  • git repository

Movie Recommendation System

CS550, Spring-2020

Implemented two of the very famous algorithms Content Based Recommendation System and Item-Item CF. Developed algorithms from scratch. We were given dataset of MovieLens which had ~9700 movies and total ~100k reviews. Used genres of a movie as feature. Given particular userID, program should reccomend top-10 movies to user. Got RMSE of 0.91 and 1.04 respectively. This project was part of CS550 coursework.

Also developed UI, where you give userID and server will return 10 movies with highest similarity and those will be recommended to user. Output will be rendered within 2 seconds. (Output demo can be found in GIT repo under outcome folder.)

Technologies used:

  • Algorithms and API developed in Python.
  • UI built using HTML, Javascript.
  • Connection between front-end and back-end using AJAX.
  • git repository
